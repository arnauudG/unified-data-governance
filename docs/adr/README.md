# Architecture Decision Records (ADR)

This directory contains Architecture Decision Records for the Unified Data Governance Platform. ADRs capture important design decisions and their context.

## Format

Each ADR is a markdown file named `NNNN-short-title.md` (e.g. `01-record-architecture-decisions.md`). Use the template below for new ADRs.

## Index

| ADR   | Title                               |
|-------|-------------------------------------|
| 0001  | Record architecture decisions       |

## Template for New ADRs

Copy this into a new file `docs/adr/NNNN-your-title.md`:

```markdown
# NNNN. Your Decision Title

**Status:** Proposed | Accepted | Deprecated | Superseded by [NNNN](NNNN-other.md)  
**Date:** YYYY-MM-DD  
**Deciders:** (optional)

## Context

What is the issue or opportunity? What constraints and forces are at play?

## Decision

What did we decide? Be specific.

## Consequences

### Positive
- ...

### Negative
- ...

### Neutral
- ...
```
